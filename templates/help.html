Help

In order to use GTS notifier, you will first need to register accounts at the pokemon global link and pushover. You will also need to make sure your pokemon GL GTS trades are visible. From the account settings, select "change privacy settings" and make sure "Global Trade Station" is set to "Everyone".

You will need to supply the submit form with your Pokemon GL profile ID and your pushover app API key. Your GL profile ID can be found on the header of the site after logging in, in the form S-000-8108-Z. Your pushover app API key can be found on the pushover main page.
